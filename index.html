My Project

Agri Robot for Multipurpose Applications

This project involves wireless Wi-Fi communication between an Android phone and the necessary components for building a robot: Wi-Fi Module (ESP 8266), Motors, Connectors, Batteries, Four Wheels, and Arduino programming. It enables a computer to control a robot, and thus, a robot can be thought of as an electromechanical machine. We developed an Android application that uses remote buttons to guide the RC machine's motion. Wi-Fi serves as the basis for communication between the controller and Android, utilizing the USART protocol.
